Pour lancer le programme de test, il suffit d'éxécuter le main de la
classe "Test.java" situé dans le package "application.tests".

L'archive est importable sous Eclipse.

Le test va charger le script "test.dtm" situé à la racine du projet, puis
générer un modèle à partir de celui-ci, finalement le programme va 
proposer un rendu Graphics2D (une fenêtre s'ouvre), puis un rendu SVG (
un fichier est généré à la racine du projet, on peut l'ouvrir avec un 
navigateur web.
